{% trans "How is CoinSafe Different from Using the Bitcoin Wallet on My Phone to Trade with Customers?" %}

{% blocktrans %} You could always use something like Mycelium Local Trader or Local Bitcoins, but there are a number of advantages to using CoinSafe: {% endblocktrans %}

  • {% blocktrans %} No more messaging with prospective customers. We're only for use in-person. Customers just show up at your store when they want to buy or sell bitcoin. If you give us your address we can post your prices online to get you additional customers. {% endblocktrans %}
  • {% blocktrans %} Ridiculously easy to use. The cashier at your store using the app doesn't have to know anything about bitcoin. It's designed so that both the cashier and the customer can be complete bitcoin newbies and it will walk them through everything. You don't have to worry about your cashiers running away with all of your bitcoins, double-spend attacks, if a bitcoin address is valid, managing private keys, etc. {% endblocktrans %}
  • {% blocktrans %} Better prices. Because it's so much more safe and convenient for the customer to go to a reputable business and not have to send messages back and forth, you can often command prices that are more favorable to you. {% endblocktrans %}

{% trans "Does CoinSafe Have My Bitcoin Private Keys?" %}

{% trans "No." %}

{% blocktrans with coinbase_url='https://coinbase.com/' blockchain_url='https://blockchain.info/' bitstamp_url='https://www.bitstamp.net/' %} If you want to sell bitcoin to customers (for cash), CoinSafe has to have a way to send the funds once the transaction is complete, so for those businesses we do this via API. The way this works is that you grant us API access to your Coinbase, blockchain.info, or Bitstamp wallet. For security, we always recommend keeping low balances on third party bitcoin sites. Remember that you always have absolute power to unlink a third party wallet from CoinSafe by revoking the credential. {% endblocktrans %}

{% blocktrans %} If all you want to do is buy bitcoin from customers (with the cash in your cash register), then all CoinSafe needs is a valid bitcoin address for customers to send your funds to. That can be any address and we won't even know who holds the private key or what software/service you used to generated that address. {% endblocktrans %}

{% trans "Can I Accept Credit Cards for Bitcoin Purchases?" %}

{% blocktrans %} You can, but we strongly recommend against it. Besides the transactions fees they charge, there is a very high risk of chargeback. Someone could use a stolen credit card (or use their own card and later claim it was stolen) and then your payment processor might take that money back from you weeks later. Remember that bitcoin transactions are irreversible, so if that happens you would have lost both the money and the bitcoin. Yikes! {% endblocktrans %}

{% trans "Why is there a Different Deposit Address for Each Customer that Sends me BTC?" %}

{% blocktrans %} In order to uniquely identify each bitcoin transaction, we generate a new bitcoin address every time a customer sends bitcoin to your business. That way, we're able to know for sure who sent you that bitcoin. We forward these transactions to your destination address automatically, and there is no fee to you at all (we cover the bitcoin network fee when forwarding the transaction). As always, you can verify this on the bitcoin blockchain for yourself! {% endblocktrans %}

{% blocktrans %} Usually, our automated system forwards the bitcoin immediately, but sometimes there can be a slight delay. We're working on improving this. If your transaction is delayed, please don't hesitate to contact us. No transaction has ever been lost, but sometimes our automated system can fall slightly behind. {% endblocktrans %}

{% trans "Who Pays the Bitcoin Network Transaction Fee?" %}

{% blocktrans with tx_fee_url='https://en.bitcoin.it/wiki/Transaction_fees' %} Contrary to popular belief, all bitcoin transactions require a very small fee in order to process in a reasonable amount of time. Currently, bitcoin's standard transaction fee is approximately 0.0001 BTC (0.1 mBTC). You can read more about this here. {% endblocktrans %}

{% blocktrans with off_chain_url='https://en.bitcoin.it/wiki/Off-Chain_Transactions' %} When you are buying bitcoin with cash, this means that the store will pay the associated transaction fee so that you receive the exact amount of bitcoin you were quoted. CoinSafe will handle this case automatically. In the event both the customer and the store are using Coinbase, Coinbase can sends the funds off the blockchain which has no transaction fees. This will happen automatically if the customer puts in an email address to send bitcoin to. {% endblocktrans %}

{% blocktrans %} When you are selling bitcoin for cash, this means that you have to pay a sufficient fee for your transaction to be processed by the bitcoin network (just like any other bitcoin transaction). The exchange rate is calculated using the amount the store received (the amount after the bitcoin network fee). Your wallet software should handle fees for you automatically, but please note that a store cannot give you cash for your bitcoin until they have received your bitcoin. If you use a non-standard fee that is too low, it's possible your transaction will take many hours to be confirmed by the network (or in extreme cases never be confirmed by the bitcoin network). Our software automatically notifies the store to not release any cash until your transaction is confirmed by the bitcoin network. In other words, save yourself a lot of headache by paying the standard fee on your bitcoin transactions to be sure they are processed by the bitcoin network (and in a reasonable amount of time). {% endblocktrans %}

{% blocktrans %} We feel this is the de facto standard in bitcoin transactions and the most fair solution for all parties. Please let us know if you disagree! {% endblocktrans %}
