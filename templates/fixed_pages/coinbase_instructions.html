{% trans "Coinbase API Instructions" %}

{% blocktrans %} Connecting your Coinbase account to CoinSafe allows us to: {% endblocktrans %}

  • {% blocktrans %} Lookup your bitcoin address at Coinbase so that shoppers can send funds there. We can also generate new addresses for you. {% endblocktrans %}
  • {% blocktrans %} Send Bitcoin from your Coinbase account to shoppers (once you confirm they have paid you in cash for it). {% endblocktrans %}
  • {% blocktrans %} Automatically buy or sell bitcoin with Coinbase using the bank account you already have linked up with them (coming soon). {% endblocktrans %}

{% blocktrans %} Please follow these instructions to grant CoinSafe access to your Coinbase account: {% endblocktrans %}

  1. {% blocktrans with coinbase_url='https://coinbase.com/settings/api' %} Go to Coinbase's API Access Page. You may be asked to login with your Coinbase credentials. {% endblocktrans %}
  2. {% blocktrans with generate_key_text="New API Key" %} Click {{ generate_key_text }}. You may be asked to enter a two factor authentication phone code if you have that configured for your account: {% endblocktrans %}
  3. {% blocktrans with select_all_text="all" accounts_text="Accounts" permisions_text="Permission" allowed_ip_text="Allowed IP Addresses" create_text="Create" %} Check the box for {{ select_all_text }} under {{ accounts_text }} and {{ permisions_text }} and leave {{ allowed_ip_text }} blank, then hit {{{{ create_text }}. {% endblocktrans %}
  4. {% blocktrans with activate_name="Enable" %} You've created API credentials, but they aren't activated yet. Now click the {{ activate_name }} button to activate them. {% endblocktrans %}
  5. {% blocktrans with coinbase_email="contact@coinbase.com" api_code_text="API verification code" %} Check your email and copy the {{ api_code_text }}. If you don't see the email, check your spam folder for a message from {{ coinbase_email }}. {% endblocktrans %}
  6. {% blocktrans with coinbase_email="contact@coinbase.com" %} Activate the API credential by pasting it onto Coinbase. {% endblocktrans %}
  7. {% blocktrans with key_field="Key" %} Your API key is now active! To get it, click on the {{ key_field }} in blue. {% endblocktrans %}
  8. {% blocktrans with key_field="API Key" secret_field="API Secret" %} Copy the {{ key_field }} and {{ secret_field }} to CoinSafe. {% endblocktrans %}

{% blocktrans %} You're all set! You can verify that your credentials are working by clicking the refresh button on CoinSafe anytime. {% endblocktrans %}
